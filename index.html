사용자에게 추천 검색어를 제공하는 상품 검색 웹 애플리케이션
개인 프로젝트
기획 및 개발
2021.09. ~ 2021.12.
'추천 키워드 검색 웹 애플리케이션'은 사용자에게 추천 키워드를 제공하는 상품 검색 웹 애플리케이션입니다. Java 8 과 Python, Spring Boot, Thymeleaf 등을 활용하여 개발되었으며, TF-IDF 기반의 코사인 유사도 알고리즘을 활용하여 추천 키워드를 제공합니다. 검색 바에서 키워드를 입력하면 자동완성으로 추천 키워드가 나타나며, 상품 리스트 페이지에서도 추천 키워드 영역에서 추천을 제공합니다.
  • 기술 스택
    • 주요 언어: Java 8, Python 3
    • 프레임워크: Spring Boot 2.5.4.
    • 프론트 엔드: HTML, CSS, JavaScript, Jquery
    • 뷰 템플릿 엔진: ThymeLeaf
    • 데이터베이스: MySQL, JPA
    • 머신 러닝 라이브러리: Pandas, scikit-learn
    • 기타: Lombok, Gradle, JEP, 네이버 쇼핑 검색 API
  • 기능과 특징
    • 네이버 쇼핑 검색 API를 활용한 상품 검색 기능: 사용자가 원하는 상품을 검색할 수 있습니다.
    • TF-IDF 기반의 코사인 유사도를 활용한 키워드 추천 알고리즘: 사용자의 검색어와 관련하여 추천 검색어를 제공합니다.
    • 프로젝트에서는 머신 러닝 기술인 TF-IDF 기반의 코사인 유사도를 활용하여 검색어와 관련된 추천 검색어를 제공합니다. 이를 통해 사용자들에게 더욱 정확하고 유용한 검색어를 제공하는 특징을 갖고 있습니다.
  • 역할과 기여
    • 개인 프로젝트로 모든 부분에 기여하였으며, 모든 기능의 설계와 개발을 진행하였습니다.
  • 결과 및 성과
    • 상품 검색 기능과 추천 검색어 기능을 모두 구현하였습니다.
    • 사용자들은 원하는 상품을 빠르게 검색하고, 추천 검색어를 통해 더욱 다양한 상품을 발견할 수 있어 검색 경험이 향상되었습니다.
  • 어려웠던 점과 극복 방법
    • JVM 특성 상 Java와 Python을 같이 실행하는 것에 어려움이 많았습니다.
    • Java Embedded Python 라이브러리를 활용하여 JVM 위에서 Python을 사용하여 문제를 해결하였습니다.
  • 배운 점과 성장
    • 프로젝트를 통해 JVM 환경에서 Python 사용을 가능하게하는 기술을 학습하였습니다. 이를 통해 다양한 언어를 융합하여 더욱 다양하고 유연한 개발이 가능해졌습니다.

  • 기술 스택
    • 주요 언어: Java 8

    • GUI 개발: Swing Component
  • 기능과 특징
    • 사용자 인증 및 계정 관리: 사용자들은 자신의 계정을 생성하고 로그인하여 프로그램을 사용할 수 있습니다.

    • 프로그램은 사용자가 등록한 선수와 전체적인 전술을 한눈에 볼 수 있도록 시각화되어 있습니다.
  • 역할과 기여

무인 열 감지 카메라
코로나 팬데믹으로 방역에 투입되는 인원 축소를 위해 무인으로 코로나 의심 환자를 선별하는 기기
팀 프로젝트
팀장, 기획 및 개발
2020.09. ~ 2020.12.
'무인 열 감지 카메라'는 Arduino 를 기반으로 개발된 프로젝트로, 코로나 의심 환자를 무인으로 선별하는 기기입니다. 초음파 센서와 적외선 카메라를 이용하여 일정 거리에 위치한 사람의 얼굴 온도를 측정합니다. 디스플레이를 활용하여 측정 결과에 따른 행동 양식 문구를 출력하여 사용자에게 안내합니다.
  • 개발 언어 및 기술
    • 개발 언어: Arduino
    • 사용 부품: AMG8833(적외선 카메라), HC-SR04(초음파 센서), ILI9341 2.2” TFT LCD(디스플레이), NodeMCU
  • 기능과 특징
    • 초음파 센서를 활용한 거리 측정: 정확한 발열 판단을 위해 사람과 기기 간 거리를 측정합니다.
    • 적외선 카메라를 활용한 체온 측정: 환자의 체온을 적외선 카메라로 측정하여 발열 여부를 판단합니다.
    • 디스플레이를 활용한 영상 송출 및 행동 양식 문구 출력: 적외선 카메라로 측정한 영상과 체온 결과를 디스플레이에 출력하고, 행동 양식 문구를 제시하여 사용자에게 안내합니다.
  • 역할과 기여
    • 프로젝트 팀장을 담당하였습니다.
    • 프로젝트의 기획과 설계부터 구현까지 모든 부분에 관여하여 프로젝트를 진행하였습니다.
  • 결과와 성과
    • 거리 또는 측정된 체온에 따라 디스플레이에 행동 양식 문구를 정확하게 출력합니다.
